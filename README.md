# AI Chatbot

An intelligent web-based chatbot developed using modern web technologies to simulate human-like conversations. The application provides users with an interactive chat interface capable of answering questions and responding to user input in real time.

---

## Overview

AI Chatbot is designed to demonstrate the fundamentals of conversational AI and chatbot development. It combines an intuitive user interface with backend processing to provide an engaging conversational experience.

The project showcases frontend development, API integration, and interactive web application design while providing a foundation for building more advanced AI-powered assistants.

## Installation

### Clone the repository

```bash
git clone https://github.com/logeshwara554/AI_chatbot.git
```

### Navigate to the project

```bash
cd AI_chatbot
```

---

## Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```

---

## How It Works

1. Launch the application.
2. Enter a question or message.
3. The chatbot processes the request.
4. The AI generates a response.
5. The reply is displayed in the chat interface.
